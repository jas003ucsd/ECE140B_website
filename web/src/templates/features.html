Features vs Benefits Part 1

The main feature would be the app. The app allows as a access point to the user inventory, grocery list, etc. The benefit to this feature is access to your inventory anywhere that you phone has service.

Feature is a low profile out of sight camera. The benefits to the camera is that it this allows for hand free tracking of items in and out of the fridge or cabinet.

Cloud storage would be another feature. This would provide the user this backup of all inventory, shopping list, etc.

Internet connectivity / Internet of Things device would a feature. This Feature would allow benefits addition services like auto shipment.

Computer vision item tracking would be last feature. This provides the benefit of know exactly what is in your inventory and save money by not buying items that are not needed. also prevent addition trips to the grocery store.

Product Interactions Part 2

The main interaction design would be Visual. We plan on a simple design that loads an easily read list that display the item, how many, and location. This data will automatically load and be present to the user immediately after opening the app.

another interaction would be Words. The data that is load will be easily filter with a search bar. This search bar will activly filter data providing intended data.

In a full product objects would be added as well. This would be done by having voice search to filter the list, order groceries, etc.
