Automatic Grocery Inventory

Have you been to the grocery store forget if you had milk? Well, now you have no worries, with the automatic Grocery Inventory. This product will track what items that you have put in the refrigerator or cabinets. This inventory can be essay access by an app.

This works by having a camera that is attached to the entrance to the groceries' storage area. When the cabinet is opened the camera is takes an image of the product entering. This image is processed with computer vision to identify the product. This item is entered into a database. The user than will be able to check what items are in the inventory. This data will then be able to retrieve from the app.
